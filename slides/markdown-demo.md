##  Usage Notes

From the terminal, pop in:

  ```yo reveal:slide "Slide Title" --markdown --attributes --notes```

Available options:

 ```--markdown --attributes --notes```

eg:

```yo reveal:slide "Slide Title" --markdown --attributes --notes```

    &lt;section id="show-a-link">
</code></pre>
